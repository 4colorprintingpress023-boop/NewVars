## 📸 ছবি আপলোড করার ধাপ

### ধাপ ১: ছবি প্রস্তুত করুন
- ফরম্যাট: JPG বা PNG
- আকার: 600x400px (পোর্টফোলিও আইটেমের জন্য)
- ফাইল সাইজ: 100-200KB

### ধাপ ২: ছবি আপলোড করুন
1. ম্যানেজমেন্ট ইউআই খুলুন
2. "Code" প্যানেল খুলুন
3. শীর্ষে "Upload" বোতাম খুঁজুন
4. আপনার ছবি নির্বাচন করুন
5. আপলোড সম্পন্ন হওয়ার জন্য অপেক্ষা করুন

### ধাপ ৩: URL কপি করুন
- আপলোড করা ছবির URL কপি করুন
- এটি দেখাবে: `https://d2xsxph8kpxj0f.cloudfront.net/...`

### ধাপ ৪: কোডে পেস্ট করুন
- উপরের কোডে `https://আপনার-ছবির-URL` এর জায়গায় পেস্ট করুন

## 💡 টিপস

1. **ছবির URL**: সবসময় `https://` দিয়ে শুরু হওয়া সম্পূর্ণ URL ব্যবহার করুন
2. **বাংলা টেক্সট**: সব জায়গায় বাংলা টেক্সট ব্যবহার করতে পারেন
3. **ইমোজি**: সেবা আইকনের জায়গায় ইমোজি ব্যবহার করুন
4. **বর্ণনা**: প্রতিটি প্রজেক্টের জন্য অনন্য এবং আকর্ষণীয় বর্ণনা লিখুন

## ❓ সমস্যা সমাধান

### ছবি দেখা যাচ্ছে না
- URL সঠিক কিনা চেক করুন
- ছবি ফরম্যাট JPG বা PNG কিনা নিশ্চিত করুন
- ব্রাউজার রিফ্রেশ করুন (Ctrl+Shift+R)

### টেক্সট ভুল দেখাচ্ছে
- বাংলা টেক্সট সঠিকভাবে কপি করেছেন কিনা চেক করুন
- কোটেশন মার্ক ঠিক আছে কিনা নিশ্চিত করুন

### পরিবর্তন সংরক্ষিত হচ্ছে না
- ভিজ্যুয়াল এডিটর ব্যবহার করলে স্বয়ংক্রিয়ভাবে সংরক্ষিত হয়
- কোড এডিটর ব্যবহার করলে Ctrl+S চাপুন
