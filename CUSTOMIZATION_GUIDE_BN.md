# আল-আমিন পোর্টফোলিও ওয়েবসাইট - সম্পূর্ণ ব্যবহারকারী গাইড

আপনার পোর্টফোলিও ওয়েবসাইট ব্যবহার করার জন্য এটি একটি সম্পূর্ণ গাইড। এখানে আমরা কভার করব কীভাবে কন্টেন্ট যোগ করবেন, ডিজাইন পরিবর্তন করবেন এবং ওয়েবসাইট প্রকাশ করবেন।

## ১. আপনার কন্টেন্ট যোগ করা

### ১.১ ব্যক্তিগত তথ্য পরিবর্তন করা

**ফাইল:** `client/src/pages/Home.tsx`

খুঁজুন এবং পরিবর্তন করুন:


## ২. ডিজাইন কাস্টমাইজ করা

### ২.১ রঙ পরিবর্তন করা

**প্রধান রঙগুলি:**

- **প্রাথমিক রঙ (নীল-কালো):** `#0a0e27`
- **সোনা অ্যাকসেন্ট:** `#d4a574`
- **সোনা হোভার:** `#c49563`

**রঙ পরিবর্তন করতে:**

ফাইল: `client/src/index.css`

খুঁজুন এবং পরিবর্তন করুন:

```css
:root {
  --primary: #0a0e27;  /* এটি পরিবর্তন করুন */
  /* অন্যান্য রঙ... */
}
```

**উদাহরণ - লাল থিমে পরিবর্তন করা:**

```css
:root {
  --primary: #1a0000;  /* গভীর লাল */
  /* অন্যান্য রঙ... */
}
```

তারপর `Home.tsx` ফাইলে সোনার রঙ পরিবর্তন করুন:

- `#d4a574` কে আপনার নতুন অ্যাকসেন্ট রঙ দিয়ে প্রতিস্থাপন করুন
- `#c49563` কে হোভার রঙ দিয়ে প্রতিস্থাপন করুন

- **বডি টেক্সট:** Poppins (আধুনিক, পরিষ্কার)

## ৩. ছবি আপলোড করা এবং ব্যবহার করা

### ৩.১ নতুন ছবি আপলোড করা

আপনার ছবি আপলোড করতে:

1. **ম্যানেজমেন্ট ইউআই খুলুন** - ডান পাশে "কোড" প্যানেল দেখুন
2. **ফাইল আপলোড করুন** - আপনার ডিজাইনের ছবি যোগ করুন
3. **URL পান** - আপলোড করা ছবির URL কপি করুন
4. **HTML-তে ব্যবহার করুন** - উপরের উদাহরণ অনুযায়ী URL ব্যবহার করুন

### ৩.২ ছবির আকার অপ্টিমাইজ করা

সেরা পারফরম্যান্সের জন্য:
- **হিরো ছবি:** 1920x1080px বা বড়
- **পোর্টফোলিও ছবি:** 600x400px
- **ফাইল সাইজ:** 100-200KB এর মধ্যে

## ৪. ওয়েবসাইট প্রকাশ করা (লাইভ করা)

### ৪.১ প্রথম প্রকাশনা

1. **পরিবর্তনগুলি সংরক্ষণ করুন** - সব ফাইল সম্পাদনা শেষ করুন
2. **ম্যানেজমেন্ট ইউআই খুলুন** - ডান পাশে প্যানেল দেখুন
3. **"Publish" বোতাম ক্লিক করুন** - শীর্ষ ডানে
4. **ডোমেইন নির্বাচন করুন** - আপনার ওয়েবসাইটের জন্য একটি ডোমেইন চয়ন করুন

### ৪.২ কাস্টম ডোমেইন সংযুক্ত করা

আপনার নিজস্ব ডোমেইন ব্যবহার করতে:

1. **ম্যানেজমেন্ট ইউআই খুলুন**
2. **সেটিংস → ডোমেইন** এ যান
3. **নতুন ডোমেইন যোগ করুন** বা ক্রয় করুন
4. **DNS সেটিংস আপডেট করুন** (যদি বিদ্যমান ডোমেইন হয়)

### ৪.৩ প্রতিটি পরিবর্তনের পরে আপডেট করা

যখনই আপনি পরিবর্তন করেন:

1. **ফাইল সংরক্ষণ করুন** (Ctrl+S বা Cmd+S)
2. **প্রিভিউ দেখুন** - ডান পাশে লাইভ প্রিভিউ দেখুন
3. **পরিবর্তন সন্তুষ্ট হলে Publish করুন**

## ৫. সাধারণ সমস্যা এবং সমাধান

### সমস্যা: ছবি দেখা যাচ্ছে না

**সমাধান:**
- ছবির URL সঠিক কিনা চেক করুন
- ইন্টারনেট সংযোগ চেক করুন
- ব্রাউজার ক্যাশ পরিষ্কার করুন (Ctrl+Shift+Delete)

### সমস্যা: ওয়েবসাইট খুবই ধীর

**সমাধান:**
- ছবির আকার কমান (100-200KB এর মধ্যে রাখুন)
- অপ্রয়োজনীয় ছবি সরান
- ব্রাউজার ক্যাশ পরিষ্কার করুন

### সমস্যা: মোবাইলে ভুলভাবে দেখা যাচ্ছে

**সমাধান:**
- ব্রাউজার ডেভেলপার টুলস খুলুন (F12)
- মোবাইল ভিউ চালু করুন
- বিভিন্ন স্ক্রিন সাইজে পরীক্ষা করুন

## ৬. উন্নত কাস্টমাইজেশন

### ৬.১ নতুন পেজ যোগ করা

নতুন পেজ তৈরি করতে:

1. **নতুন ফাইল তৈরি করুন:** `client/src/pages/YourPage.tsx`
2. **কন্টেন্ট যোগ করুন**
3. **App.tsx এ রুট যোগ করুন:**

```javascript
import YourPage from "./pages/YourPage";

<Route path="/your-page" component={YourPage} />
```

### ৬.২ নেভিগেশন আপডেট করা

নেভিগেশন মেনুতে নতুন লিঙ্ক যোগ করতে, `Home.tsx` এ খুঁজুন:

```javascript
<a href="#portfolio" className="text-foreground hover:text-[#d4a574]">
  Portfolio
</a>
```

নতুন লিঙ্ক যোগ করুন:

```javascript
<a href="#your-section" className="text-foreground hover:text-[#d4a574]">
  আপনার সেকশন
</a>
```

### ৬.৩ ফর্ম কার্যকর করা

যোগাযোগ ফর্মটি বর্তমানে ভিজ্যুয়াল। ইমেইল পাঠানোর জন্য ব্যাকএন্ড সেটআপ প্রয়োজন।

আমাদের কাছে ফিরে আসুন যদি আপনি এটি সক্ষম করতে চান।

## ৭. টিপস এবং ট্রিকস

1. **সব লিঙ্ক পরীক্ষা করুন** - সবকিছু কাজ করছে কিনা
